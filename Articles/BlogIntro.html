An this is my blog

May 18, 2022

Ok, but why?

I will be honest, I didn't know how to write a blog. I was looking for a way to write a blog and I found a template on w3schools and from there I started to write my own blog.

So, after grabbing the example code I started modifying it to fit my needs. I made the background purple, I made the cards a grey shade, which is just color: rgb(52, 52, 52); in CSS. I also added a little bit of text to the cards, rounded corners using border-radius: 15px; and I added a resource directory for the images. Presumably, the project logo and screenshots of Visual Studio Code.

I also added a link to the GitHub repository, which is the same as the one on the header.

The most fun part were the About Me section which you are seeing sticking on the right side of the page, the github integration and the forms at the end of this very article.

The left side of the page is the blog, is hold on this position while scrolling using position: sticky; applied on the whole colum on this table.

The github integration is done using the code from an open source project called emgithub which uses MIT license, so we should be able to use it.

The forms are done using something called textarea tag which can be scrolled vertically.
